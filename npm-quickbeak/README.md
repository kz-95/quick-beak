# QuickBeak

Single-file, offline-first, privacy-first issue tracker for indie developers and makers.

Drop bugs, suggestions, and feedback as you build. A local AI cleans up your notes
(text never leaves your machine), and everything syncs through a Google Sheet you own.
No server, no subscription.

## Status

Published to npm as a name reservation. The app is in active development; this
package does not yet ship the application.

```
npm install quickbeak
```
