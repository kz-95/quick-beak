# QuickBeak

Single-file, offline-first, privacy-first issue tracker for indie developers.

Drop bugs, suggestions, and feedback as you build. A local AI cleans up your notes
(text never leaves your machine), everything syncs through a Google Sheet you own.

## Quick start

```bash
npx quickbeak
```

Fetches the latest QuickBeak and opens it in your browser. No install, no bloat.
